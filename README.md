# SalonEase

## 🛠 Technologies Used : 
- 🚀 **Spring Boot**  
- 🔐 **Spring Security**  
- 🔑 **JWT (JSON Web Token)**  
- 🗃️ **Spring Data JPA**  
- 🐬 **MySQL** 
- ⚡ **Lombok**  
- 📦 **Maven**
- 🏗️ **Microservice Architecture**
