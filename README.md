# SalonEase

## 🛠 Technologies Used : 
🚀 **Spring Boot**  
🔐 **Spring Security**  
🔑 **JWT (JSON Web Token)**  
🗃️ **Spring Data JPA**  
🐬 **MySQL** *(or H2 for an in-memory database)*  
⚡ **Lombok**  
📦 **Maven**
**Microservice Architecture**

---

## ⚙️ Installation and Setup
### 📌 Prerequisites
Ensure you have the following installed before setting up the project:
- **Java 17+** ☕
- **Maven** 🏗️
- **MySQL** *(optional, if not using H2)* 🗄️
